# 气动步兵Pneumatic_Robot
这个库来自于我来说就是我对我自己大一时候想法进行重新构成。
22年的时候云浩说过这几年的发射机构一直没什么变化，常规的机构都是摩擦轮构成，基本没什么特别大的创新，但是对我来说气动才是真正的未来，一套车上两套系统确实不太可行，但是在不存在下坠的前提下我觉得这才是我真正想要的。。。
